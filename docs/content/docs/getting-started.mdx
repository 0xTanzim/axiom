# Getting Started

Create your first Axiom application in under 5 minutes.

## Prerequisites

<Callout type="info" title="Java Version">
Axiom requires **Java 21 or later** (virtual thread support).

- **Minimum:** Java 21
- **Recommended:** Java 25 LTS

**Download:**
- Windows: [Adoptium OpenJDK](https://adoptium.net/temurin/releases/?os=windows&arch=x64&package=jdk&version=21) (MSI installer)
- macOS: `brew install openjdk@21` or [Download](https://adoptium.net/temurin/releases/?os=mac&arch=x64&package=jdk&version=21)
- Linux: `sudo apt install openjdk-21-jdk` or [Download](https://adoptium.net/temurin/releases/?os=linux&arch=x64&package=jdk&version=21)
</Callout>

## Quick Start with Maven Archetype

The fastest way to create a new Axiom project (works on all platforms):

<Tabs items={['One-line (Unix/Mac)', 'Multi-line (Windows)', 'Interactive']}>
  <Tab value="One-line (Unix/Mac)">
```bash

  -DarchetypeGroupId=io.github.0xtanzim \
  -DarchetypeArtifactId=axiom-quickstart \
  -DarchetypeVersion=0.1.3 \
  -DgroupId=com.example \
  -DartifactId=my-app \
  -Dversion=1.0.0 \
  -DinteractiveMode=false
```

## Manual Installation

Quick setup for existing projects:

<Tabs items={['Maven', 'Gradle Groovy', 'Gradle Kotlin']}>
  <Tab value="Maven">
```xml title="pom.xml"
<dependencies>
    <dependency>
        <groupId>io.github.0xtanzim</groupId>
        <artifactId>axiom</artifactId>
        <version>0.1.3</version>
    </dependency>

    <!-- SLF4J Logging Implementation (recommended) -->
    <dependency>
        <groupId>org.slf4j</groupId>
        <artifactId>slf4j-simple</artifactId>
        <version>2.0.16</version>
    </dependency>
</dependencies>

<properties>
    <maven.compiler.source>21</maven.compiler.source>
    <maven.compiler.target>21</maven.compiler.target>
</properties>
```
  </Tab>
  <Tab value="Gradle Groovy">
```groovy title="build.gradle"
plugins {
    id 'java'
    id 'application'
}

java {
    toolchain {
        languageVersion = JavaLanguageVersion.of(21)
    }
}

dependencies {
    implementation 'io.github.0xtanzim:axiom:0.1.3'
}

application {
    mainClass = 'App'
}
```
  </Tab>
  <Tab value="Gradle Kotlin">
```kotlin title="build.gradle.kts"
plugins {
    java
    application

java {
    toolchain {
        languageVersion.set(JavaLanguageVersion.of(21))
    }
}

dependencies {
    implementation("io.github.0xtanzim:axiom:0.1.3")

    mainClass.set("App")
```

## Add More Routes

```java title="App.java"
import io.axiom.core.app.Axiom;
import io.axiom.core.routing.Router;
import java.util.List;
import java.util.Map;

public class App {
    public static void main(String[] args) {
        Router router = new Router();

        // Text response
        router.get("/", c -> c.text("Hello!"));

        // JSON response
        router.get("/api/status", c ->
            c.json(Map.of("status", "ok", "version", "0.1.3"))
        );

        // Path parameters
        router.get("/users/:id", c -> {
            String id = c.param("id");
            c.json(Map.of("id", id, "name", "User " + id));
        });

        // Query parameters
        router.get("/search", c -> {
            String query = c.query("q");
            c.json(Map.of("query", query, "results", List.of()));
        });

        // POST with body
        router.post("/users", c -> {
            User user = c.body(User.class);
            c.status(201);
            c.json(user);
        });

        Axiom.start(router, 8080);
    }
}

record User(String name, String email) {}
```

## Full Control with Middleware

When you need middleware, lifecycle hooks, or error handling:

```java title="App.java"
import io.axiom.core.app.*;
import io.axiom.core.routing.Router;
import java.util.Map;

public class App {
    public static void main(String[] args) {
        Router router = new Router();
        router.get("/", c -> c.text("Hello!"));
        router.get("/users/:id", c -> c.json(Map.of("id", c.param("id"))));

        App app = Axiom.create();

        // Middleware (choose your style)
        app.use((ctx, next) -> {
            System.out.println(ctx.method() + " " + ctx.path());
            next.run();
        });

        // Lifecycle hooks
        app.onStart(() -> System.out.println("Connecting to database..."));
        app.onReady(() -> System.out.println("ðŸš€ Server ready!"));
        app.onShutdown(() -> System.out.println("Cleaning up..."));

        // Error handling
        app.onError((ctx, e) -> {
            ctx.status(500);
            ctx.json(Map.of("error", e.getMessage()));
        });

        app.route(router);
        app.listen(8080);
    }
}
```
